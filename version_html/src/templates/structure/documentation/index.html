Installing Environment


  1. Install Node.js 7.5.0 (https://nodejs.org)
  2. Install Gulp and Bower npm install --global bower gulp

Static version serving


  1. cd version_html/ enter to static version folder
  2. npm install install node plugins (once)
  3. bower install install vendors plugins (once)
  4. gulp for live serving

Angular version serving


  1. npm install -g @angular/cli install Angular CLI globally
  2. cd version_angular/ enter to angular version folder
  3. npm install install node plugins (once)
  4. ng start for live serving
  5. ng build --prod --aot for production build (/dist/ folder)
